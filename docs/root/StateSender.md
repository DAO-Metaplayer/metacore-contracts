### syncState

```solidity
function syncState(address receiver, bytes data) external nonpayable
```

Generates sync state event based on receiver and data. Anyone can call this method to emit an event. Receiver on Metachain should add check based on sender.



#### Parameters

| Name | Type | Description |
|---|---|---|
| receiver | address | Receiver address on Metachain |
| data | bytes | Data to send on Metachain |
